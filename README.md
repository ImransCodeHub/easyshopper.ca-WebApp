## Website URL: https://easyshopper.ca/

## Frontend Requirements
`npm install` 

## Backend Requirements
`npm install express`  
`npm install mongodb`

## Launching App (Development State)
`cd backend`  
`npm start dev`

`cd frontend`  
`npm start`

## Preparing App For Production
`cd frontend`  
`npm run build`  
`cp -r build ../backend`
